# Cyber Investigation Dashboard

A desktop and web-based OSINT launcher built with React, Vite, and Electron.

## Features
- One-click launcher for OSINT and cyber investigation tools
- Domain, IP, email, username, and wallet workflows
- Categories for infrastructure, people, breach, GEO, crypto, analysis, scam, and reporting
- Electron desktop packaging support

## Tech Stack
- React
- Vite
- Electron
- Lucide React

## Run Locally
```bash
npm install
npm run dev
npm run package-win
